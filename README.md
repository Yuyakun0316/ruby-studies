# Ruby学習用リポジトリ (Ruby Studies)

プログラミング言語「Ruby」の基礎理解を深めるために作成した、学習記録用リポジトリです。
テックキャンプでの学習に加え、基礎的な文法やオブジェクト指向の概念をコードを書いて実験・検証しました。

## 📚 学習した内容

### 基礎文法
- **変数・定数**: データの保持とスコープの実験 (`variable_sample.rb`)
- **条件分岐**: if文によるロジックの制御 (`conditional_sample.rb`)
- **繰り返し**: each, times, while の使い分け (`loop_types.rb`)
- **配列・ハッシュ**: データの集合と操作 (`array_each_sample.rb`, `hash_sample.rb`)

### 応用・実践
- **メソッド**: 引数と戻り値の理解 (`method_sample.rb`)
- **クラス・オブジェクト指向**: クラスの定義とインスタンス化 (`class_sample.rb`)
- **継承**: 親クラスの機能を継承する仕組み (`inheritance_sample.rb`)
- **例外処理**: begin/rescue によるエラーハンドリング (`exception_sample.rb`)
- **Gemの利用**: Bundlerを使った外部ライブラリ(`faker`)の導入 (`gem_sample.rb`)

### 成果物
- **FizzBuzz問題**: 論理的思考のトレーニング (`fizzbuzz.rb`)
- **CLIショッピングアプリ**: 基礎項目の総まとめとして作成 (`shopping_app.rb`)

## 🛠 使用技術
- Ruby 3.2.0
- Git / GitHub
- Bundler
